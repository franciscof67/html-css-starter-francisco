# Mini Landing Page – HTML/CSS Starter

Questo è un mini‑progetto creato per con Git, GitHub e la gestione di un repository professionale.

## 🚀 Tecnologie utilizzate
- HTML5
- CSS3

## 📁 Struttura del progetto
- `index.html` – struttura della pagina
- `style.css` – stile base
- `assets/` – cartella per immagini e risorse

## 🎯 Obiettivo
Creare una landing page semplice e pulita, pubblicata su GitHub con commit chiari e organizzati.
